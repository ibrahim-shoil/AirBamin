How it works

1
Connect to the same network
Wi‑Fi or phone hotspot—no internet required.
2
Open AirBamin on PC
Go to Transfer Center, refresh the link if you changed networks.
3
Pair the phone
Scan the QR or paste the link; choose Upload, Send, or Mirror.
4
Send, receive, or mirror
Watch live progress on desktop and keep the phone page open until done.

Support & questions

Is the Windows build signed? 1.2.0 is ready for distribution; if SmartScreen appears, choose “More info” → “Run anyway” while code signing is in progress.
Which version should I host on tecbamin.com? Place the latest Windows installer at /downloads/AirBamin-1.2.0.exe to match this page. Keep /downloads/airbamin-1.0.0-setup.exe for legacy users if needed.
Where do I get the mobile build? Contact us for the Android APK or iOS TestFlight invite. The site buttons link to the contact form so we can share the latest build immediately.
Need a hand? tecbamin.com/contact for support, onboarding, or to report an issue.
